# Story Assets

Place your narrative assets here:

- `tokyo-signal-poster.svg` or `.jpg/.webp`: static fallback image
- `tokyo-signal-loop.mp4`: optional short loop (6-12s, muted, seamless)

Recommended export settings:

- Video: H.264, 1080p max, 24-30 fps, target 2-4 Mbps
- Image: WebP preferred for photo/painted scenes
- Keep color contrast high enough for text overlay readability
