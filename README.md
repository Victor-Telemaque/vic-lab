# Vic Lab - Tokyo Signal Portfolio

Creative front-end portfolio built to showcase product thinking, visual craft, and scalable engineering.

## Elevator Pitch

I design and build interactive web products that balance:

- strong front-end architecture
- visual identity and motion details
- performance and maintainability

This project is the public lab behind that approach.

## Live Goals

- Attract freelance and full-time front-end opportunities
- Demonstrate product-oriented case studies
- Highlight creative execution without sacrificing code quality
- Prepare a multilingual profile for international hiring

## Tech Stack

- Next.js (App Router)
- React 19 + TypeScript
- SCSS Modules
- Framer Motion
- react-intl (`fr` / `en`)

## Key Features (V1)

- Strong hero with visual storytelling structure
- Atmospheric UI layer and subtle micro-interactions
- Case-study oriented project section (impact-driven content)
- Responsive design from mobile to desktop
- Internationalization-ready architecture with locale files in `src/locales`

## Project Structure

```text
src/
  app/
  components/
    i18n/
    media/
    sections/
  data/
  i18n/
  locales/
public/
  assets/
    story/
```

## Run Locally

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Scripts

- `npm run dev`: start development server
- `npm run lint`: run ESLint
- `npm run build`: build production bundle
- `npm run start`: run production server
